# 📊 Zerodha Kite Database Project

## 📌 Overview
This project is a fully designed relational database system that simulates a real-world stock trading platform like Zerodha.

## 🚀 Features
- User Management & KYC
- Trading Accounts & Orders
- Stock Market Data
- Portfolio Tracking
- Transactions & Payments
- Alerts & Watchlist

## 🛠️ Technologies Used
- MySQL
- SQL (Joins, Subqueries, Aggregations)

## 📂 Project Structure
- SQL Scripts
- Sample Data
- Presentation (PPT)

## 📊 Key Concepts
- Normalization
- Foreign Keys & Constraints
- GROUP BY, HAVING
- EXISTS, IN, ANY
- Real-world trading logic

## 🎯 Outcome
This project demonstrates strong SQL skills and real-world database design suitable for financial applications.
